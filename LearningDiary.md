Learning from the Project

From this project, I learned how to create an entire Android application using the Java language and the Android Studio development environment. I learned how to make activities work, create layouts using XML, switch between different screens, and apply different user interfaces such as Button, TextView, EditText, RadioButton, Spinner, and ListView.

I got the experience of developing Android projects, building them using Gradle build tools, working with Git and GitHub.

Problems I Had
1. Build Problems

I had issues related to building my app because some IDs were missing in my Java files.

2. Activities Navigation

At the beginning of working on this project, some buttons were not able to launch their activities due to wrong intent implementations.

3. Balance Updating
The home screen showed only zero because transactions could not be added to the balance.

4. Uploading to GitHub
There was some difficulty uploading my app to GitHub for the first time using Android Studio.

Solving the Problems
Build Problems
I checked all the issues mentioned in error messages in Android Studio.

Activities Navigation
All implementations of intent and activities registration in AndroidManifest.xml were checked.
Balance Updates

The logic for calculating total income, total expense, and balance was done based on the transactions.

GitHub Commit
Initial commit for Git was done, followed by connecting with GitHub and uploading the project onto GitHub.

Reflections
This project gave me an insight into how an entire Android app can be developed starting from designing the UI up to the coding and finally uploading the project to GitHub. My problem-solving abilities were enhanced, and I became more efficient with Android Studio as well as Java. In case I keep working further on this project, I will include a database for storing data permanently.
This project was a great learning experience overall.
