UniBudget Finland
Author

Anisha Tasnim

Description

UniBudget Finland is a simple Android budgeting application developed using Java and Android Studio. The application helps users track their income and expenses, view transaction history, and calculate their current balance.

Features
Add income transactions
Add expense transactions
Categorize transactions
View transaction history
Calculate total income
Calculate total expenses
Display current balance
Simple and user-friendly interface
Technologies Used
Java
Android Studio
XML Layouts
How to Run the Project
Requirements
Android Studio
Android SDK
Android device or emulator
Steps
Clone or download the repository.
Open the project in Android Studio.
Wait for Gradle synchronization to finish.
Connect an Android device or start an emulator.
Click the Run button.
The application will install and launch automatically.
Project Structure
MainActivity.java – Home screen and balance calculations
AddTransactionActivity.java – Add income and expense transactions
HistoryActivity.java – Display transaction history
AboutActivity.java – Information about the application
Future Improvements
Store data permanently using SQLite or SharedPreferences

Monthly budgeting reports
